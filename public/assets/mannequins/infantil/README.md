# Manequim de referência — Infantil

Esta pasta é usada pelo **Estúdio Criativo (IA) — Catálogo** para a **grade infantil**. As imagens aqui servem como referência fixa: a IA combina o produto com essas fotos do manequim em vez de inventar um manequim.

## O que colocar aqui

Coloque **duas imagens** com os nomes abaixo. **Formatos aceitos automaticamente:** PNG, JPG, JPEG e WEBP. O sistema procura, nessa ordem, o arquivo com cada extensão e usa o primeiro que encontrar (ex.: `manequim-infantil-frente.png` ou `manequim-infantil-frente.jpg`).

| Nome do arquivo (qualquer extensão suportada) | Descrição |
|-----------------------------------------------|-----------|
| `manequim-infantil-frente` (.png, .jpg, .jpeg ou .webp) | Foto **frente** do manequim infantil (vazio, sem roupa). Será usada para gerar a **Foto Frente** do produto vestido nesse manequim. |
| `manequim-infantil-costas` (.png, .jpg, .jpeg ou .webp) | Foto **costas** do manequim infantil (vazio, sem roupa). Será usada para gerar a **Foto Costas** do produto vestido nesse manequim. |

## Dicas para as fotos

- **Fundo:** de preferência neutro (branco ou cinza claro) para facilitar a combinação com o produto.
- **Enquadramento:** corpo inteiro do manequim, mesmo ângulo/pose que você quer nas fotos de catálogo.
- **Iluminação:** uniforme, sem sombras fortes.
- **Tamanho:** resolução adequada para catálogo (ex.: 1080px de largura ou maior).

## Uso no sistema

- Na tela **Produtos → Estúdio Criativo (IA) — Catálogo**, ao gerar **Foto Frente** e **Foto Costas** para produtos da grade **infantil**, o sistema usará estas imagens como base.
- Depois será possível adicionar pastas semelhantes para outros públicos (ex.: `adulto/`, `bebe/`) com as mesmas convenções de nome: `manequim-<publico>-frente` e `manequim-<publico>-costas`.
